## Notes for Use

- Issue a Change Order for **any** request that adds work, changes a deliverable, extends a timeline, or modifies agreed fees — no matter how small it seems.
- "One small addition" conversations should always produce a signed Change Order before work begins.
- Use the Change Order Log to maintain a running project history.
- Electronic signatures (DocuSign, Adobe Sign, etc.) are legally valid.
- Never begin changed or additional work based on email approval alone — get the Change Order signed.
