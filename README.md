# PanteraClaw.com

AI/ML Consulting Website for Pantera Claw (Agentic AI LLC)

## Overview

A high-converting, SEO/GEO-optimized consulting website showcasing AI/ML expertise including:
- **Agentic Workflow Design** - LangGraph orchestration and multi-agent architectures
- **High-Precision ML** - 98% accuracy drone classification, computer vision
- **Strategic Data Engineering** - DoD-compliant PySpark/Databricks pipelines
- **Causal Insight & Policy** - Causal inference modeling for strategic decisions

## Tech Stack

- React 18 + TypeScript
- React Router v7 (Data mode)
- Tailwind CSS v4
- Vite
- Lucide React icons
- Schema.org structured data

## Features

- Multi-page responsive design
- Schema.org JSON-LD markup for SEO
- Utah/Silicon Slopes local SEO targeting
- Mobile-first responsive navigation
- Case studies with technical depth
- Contact form
- Professional service architecture

## Getting Started

```bash
npm install
npm run dev
```

## Pages

- `/` - Homepage with hero, services overview, case study previews
- `/services` - Detailed service modules with capabilities
- `/case-study/drone-classification` - 98% accuracy drone audio classification
- `/case-study/nlp-sql` - NLP-to-SQL enterprise application
- `/about` - Leadership, credentials, Utah tech ecosystem
- `/contact` - Contact form and business information
